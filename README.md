📊 Sales Data Analysis Dashboard


🎯 Project Objectives

The main objectives of this project are:

Load sales data from CSV files

Clean and preprocess the dataset

Perform exploratory data analysis (EDA)

Calculate important sales metrics

Generate sales trends and reports

Create data visualizations using charts

🧠 What I Learned

Pandas Fundamentals
Learned how to work with DataFrames and analyze datasets.

Data Cleaning
Removed duplicate records and handled missing values.

Data Analysis
Calculated key metrics such as total sales and average order value.

Data Visualization
Created charts using matplotlib to represent sales data visually.

Report Generation
Exported analysis results into files such as Excel and images.

✨ Features

Load sales data from CSV files

Clean and preprocess the dataset

Calculate total sales and average order value

Analyze sales by product category

Generate charts for sales visualization

Export reports and charts

Simple command-line interface

⚙️ Setup and Installation
1️⃣ Install Required Libraries

Run the following command:

pip install pandas matplotlib numpy openpyxl


▶️ How to Run the Project

Run the following command:

python main.py

The program will:

Load the sales dataset

Clean the data

Calculate statistics

Analyze sales by category

Generate visual charts

Save reports and charts
